    • Hub Deployment

    On the hub through ISO deployment, we need to add a couple of shell scripts and execute a command to run both scripts and install the defender.

    We need to make an API call to the Prisma server to fetch first a JWT token. Before that we need to have an administrator account on Prisma compute to create either a service account or an API access and secret key.

    Input the access and secret key under the body of the API request. I would recommend using postman to format the request correctly and then through postman convert the request into Curl so we can run it through the command line.

    After getting the JWT token we need to save it and pass it into a second Curl command provided by Prisma to download the twistlock package and install on the single host defender.

    Echo the Curl command inputted with the new JWT token you acquired. This will download and install the twistlock defender.
